About Us

Boston Bathrooms was inspired by the got2goNYC bathroom map created by Theodora Siegel. This campaign is a community-led campaign to improve access to public restrooms, especially in more residential areas, like Allston. User generated content is used to gather information on restroom locations, which will be implemented onto an interactive map that is free to use, essentially helping everyone, from commuters to tourists.
